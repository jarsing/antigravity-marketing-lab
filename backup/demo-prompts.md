# 講者 Demo Prompts

## 1. CLI 產生 demo 主標題

```text
請讀取 challenge/demo-brief.md，
使用 local-marketing-agent Skill 執行「Stage 1：主標題產生」。

請自行檢查標題是否符合 Skill 規則；
若不符合，請修正後再交付。

最後只將通過檢查的最終主標題
儲存到 outputs/demo-headline.md。
```

## 2. Antigravity 2.0 / IDE 網站修改任務

請將下方「已核准主標題」替換成現場會眾選出的標題。

```text
這是一句已由現場會眾產生，並通過人工選用的活動主標題：

「請貼入現場選出的主標題」

請閱讀 website/ 中的現有活動宣傳網站，完成以下開發任務：

1. 將上述已核准主標題整合到首頁 Hero 區塊。
2. 新增一個明顯的 CTA 按鈕，文字為「加入 LINE 取得散步資訊」。
3. 加入一段簡短說明，讓訪客理解可以吃肉圓並順道探索附近景點。
4. 保持手機版友善與目前溫暖、地方感的視覺風格。
5. 不得自行加入折扣、贈品、抽獎、限量名額或活動成效。

完成程式修改後，請啟動網站並使用瀏覽器驗證：
- Hero 主標題是否正確顯示；
- CTA 按鈕是否清楚可見；
- 手機版畫面是否可閱讀；
- 頁面是否沒有未經提供的優惠資訊。

最後請回報修改檔案、完成功能與驗證結果。
```
